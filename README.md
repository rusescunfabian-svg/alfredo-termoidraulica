# Alfredo Termoidraulica — sito vetrina

Sito statico (HTML, CSS, JavaScript) pronto per **GitHub Pages**.

## Pubblicazione su GitHub Pages

1. Crea un nuovo repository su GitHub (es. `alfredo-termoidraulica`).
2. Carica i file della cartella `alfredo-termoidraulica` nella root del repository (oppure sposta il contenuto nella root se preferisci).
3. Su GitHub: **Settings → Pages → Build and deployment → Branch**: scegli `main` (o `master`) e cartella `/ (root)`.
4. Dopo qualche minuto il sito sarà disponibile su `https://TUO-USERNAME.github.io/NOME-REPO/`.
5. In `index.html`, aggiorna l’URL nel blocco JSON-LD (`schema.org`) sostituendo `USERNAME` e `REPO` con i tuoi valori reali.

## Modifiche locali

Apri `index.html` nel browser oppure servi la cartella con un server statico locale.
